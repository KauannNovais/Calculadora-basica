# Simple Blue Calculator

Calculadora simples em **C# com WPF**, com visual moderno em **tons de azul**, pronta para usar como projeto de portfólio ou repositório no GitHub.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Porcentagem
- Limpar tela
- Apagar último dígito
- Interface azul e organizada

## Tecnologias

- C#
- .NET 8
- WPF
- XAML

## Como executar

1. Abra o projeto no **Visual Studio 2022** ou superior.
2. Verifique se o workload de **desenvolvimento desktop com .NET** está instalado.
3. Execute o projeto.
